# springmvc-mybatis-rest-pro
project infrastructure

支持multi Datasource版本
update 2016.3.9

支持shiro版本
update 2016.3.25

香港节点有db1,db2,大陆节点有db3.
目的是让3个数据库接入同一个web服务，由spring-mybatis管理，即数据源:datasource1,datasource2,datasource3
